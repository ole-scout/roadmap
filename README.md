# OLE Scout public roadmap

In this repository, you can find the official [OLE Scout public product roadmap](https://github.com/orgs/ole-scout/projects/1). Our product roadmap is where you can learn about what features we're working on, what stage they're in, and when we expect to bring them to you.

Have any questions or comments about items on the roadmap? Use the roadmap [discussions](https://github.com/ole-scout/roadmap/discussions) to [suggest new features](https://github.com/ole-scout/roadmap/discussions/categories/ideas) or [ask general questions](https://github.com/ole-scout/roadmap/discussions/categories/q-a). Or share your feedback with us more privately using the contact form on [our website](https://ole-scout.com).

## Guide to the roadmap

Every item on the roadmap is an issue, with information indicating the following:

* A **tier** label indicating what customer tier the feature will be enabled for in the OLE Scout Cloud service (on-premise hosting is only offered at the *Enterprise* tier).

* A **release target** milestone indicating the target timeframe in which a feature is planned to become available to OLE Scout Cloud customers.

## Roadmap stages

Our development process is split into the following phases to help us better prioritize feature requests:

* *Ideas*: early draft feature proposals that have not yet been sufficiently investigated for prioritization.

* *Exploring*: feature proposals we are currently investigating or sketching early design concepts for.

* *Approved*: planned features that have been sufficiently investigated to start implementing.

* *In Development*: features that we are currently actively working on, usually with a predicted *release target*

+ *Released*: features that have been recently completed and are now available or in the process of being rolled out.

* *Postponed / Icebox*: features that have been rejected for immediate development but are still being considered for future development.

Please note that any indications about proposed or unfished features are only provided to offer transparency into our development process to our existing customers. If you need us to implement a feature or commit to providing a feature by a given date, please contact our sales team using the contact form on [our website](https://ole-scout.com) for a formal quote.
